# 📱 Простой браузерный мессенджер

Чат-приложение в браузере без регистрации на почту. Работает на ПК, планшете и телефоне.

## ✨ Возможности

- 📝 Регистрация по имени + пароль (без почты)
- 💬 Личные и групповые сообщения
- 🌐 Работает через браузер (Chrome, Firefox, Safari, Edge)
- 📡 Синхронизация через сервер
- 📴 Поддержка Bluetooth (для оффлайн режима)
- 🔒 Сообщения хранятся на сервере

## 🚀 Быстрый старт

### Локально (ПК)

```bash
pip install -r requirements.txt
python pipl.py
```

Откройте в браузере: **http://localhost:8000**

### На облаке (Render)

1. [Вилкните репо](https://github.com/YOUR_USERNAME/pipl-messenger) в свой аккаунт
2. Перейдите на [Render.com](https://render.com)
3. Создайте новый **Web Service**
4. Подключите ваш GitHub репо
5. Укажите команду: `python pipl.py`
6. Готово! Приложение откроется по ссылке типа `https://pipl-messenger-xxxxx.onrender.com`

## 📋 Структура

- `pipl.py` — основное приложение
- `requirements.txt` — зависимости Python
- `Procfile` — конфиг для облачного хостинга
- `pipl_users.json` — база пользователей (создается автоматически)

## 🔧 API Endpoints

| Метод | URL | Описание |
|-------|-----|---------|
| POST | `/api/register` | Регистрация пользователя |
| POST | `/api/login` | Вход в систему |
| POST | `/api/message` | Отправка сообщения |
| GET | `/api/poll` | Получить новые сообщения |
| GET | `/api/online` | Список онлайн пользователей |

## 💾 Данные

Сообщения и пользователи сохраняются в `pipl_users.json` локально. Для облачного хостинга рекомендуется использовать базу данных (PostgreSQL на Render доступен).

## 🛠 Требования

- Python 3.8+
- Flask 3.0+
- Браузер с поддержкой WebSocket (опционально)
